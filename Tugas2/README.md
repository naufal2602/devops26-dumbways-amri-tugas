## Pengertian
* **SH (Bourne Shell)** adalah shell UNIX dasar yang ringan dan mengikuti standar POSIX.
* **BASH (Bourne-Again Shell)** adalah pengembangan dari SH dengan fitur yang lebih lengkap dan modern.

## Perbedaan Utama
| SH                             | BASH                                                 |
| ------------------------------ | ---------------------------------------------------- |
| Minimalis dan ringan           | Lebih modern dan kaya fitur                          |
| Fokus pada command dasar       | Mendukung array, alias, history, dan auto completion |
| Lebih portabel                 | Lebih nyaman untuk scripting kompleks                |
| Lebih cepat dan hemat resource | Menggunakan resource lebih banyak                    |
| Cocok untuk script sistem      | Cocok untuk penggunaan harian dan development        |

## Kompatibilitas
* Script SH umumnya dapat berjalan di BASH.
* Script khusus BASH belum tentu dapat berjalan di SH.

## Kesimpulan
SH cocok digunakan untuk script sederhana dan kebutuhan sistem karena ringan dan stabil. Sedangkan BASH lebih cocok untuk development dan automation karena memiliki fitur yang lebih lengkap dan fleksibel.
